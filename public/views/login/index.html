Welcome to Socialized

Socialized is a technology demo for ScalaJs.io; an implementation of the Node.js and npm in Scala.js.

The goals of this project are:
  • To provide a comprehensive example of how to develop an application using ScalaJs.io.
  • To create an open source social networking portal for developers.
  • To have have fun!

Developers are welcomed

  • ScalaJs.io needs contributors to help to realize the platform vision.
  • Help to develop/maintain new/existing features in ScalaJs.io.
  • Help to add new features in the Socialize portal.
